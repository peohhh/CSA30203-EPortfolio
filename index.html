Career Exploration

Chosen Career: Network Security Analyst

A Network Security Analyst is responsible for protecting an organization’s computer network from cyber threats. This role involves monitoring network traffic, identifying suspicious activities, supporting firewall and access control configuration, and assisting in incident response.


I chose this career because I am interested in networking, cybersecurity, and protecting digital systems from threats. During my studies, I learned networking concepts, subnetting, IP addressing, cybersecurity technologies, web systems, and IT-related project development. These experiences strengthened my interest in pursuing a career in network security.


Job Responsibilities

  • Monitor network traffic and security events
  • Identify suspicious activities and possible threats
  • Support firewall and access control configuration
  • Assist in incident response activities
  • Prepare reports on network security issues

Required Skills

  • Networking fundamentals
  • TCP/IP and subnetting
  • Linux basics
  • Wireshark and packet analysis
  • Security monitoring tools
  • Communication and reporting skills

Relevance to Cybersecurity

Network security is important because organizations depend on secure networks to protect customer data, prevent cyberattacks, and maintain business continuity. As more services move online, the role of network security becomes increasingly important in modern cybersecurity.
