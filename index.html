Career Exploration

Chosen Career: Network Security Analyst

A Network Security Analyst is responsible for protecting an organization’s computer network from cyber threats. This role involves monitoring network traffic, detecting suspicious activities, supporting security tools, and responding to security incidents.


I choose this career because it matches my background as an Information Technology student and my learning experience in networking, cybersecurity, system design, and digital technology projects.


Job Responsibilities

  • Monitor network traffic and security events
  • Identify suspicious activities and threats
  • Support firewall and access control configuration
  • Assist in incident response activities
  • Prepare reports on network security issues

Required Technical Skills

  • Networking fundamentals
  • TCP/IP and subnetting
  • Linux basics
  • Wireshark and packet analysis
  • Firewall and security monitoring tools

Relevance to Modern Cybersecurity

Network security is important because organizations depend on secure networks to protect customer data, prevent cyberattacks, and maintain business continuity. As more services move online, the role of network security becomes increasingly important.
