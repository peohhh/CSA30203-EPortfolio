Career Exploration

Chosen Career: Network Security Analyst

A Network Security Analyst is responsible for protecting an organization’s computer network from cyber threats. This role involves monitoring network traffic, detecting suspicious activities, managing security tools, and responding to security incidents.


I choose this career because it matches my background in Internet Computing, networking, cybersecurity assignments, and my interest in protecting digital systems.


Job Responsibilities

  • Monitor network activity
  • Detect cyber threats
  • Respond to incidents
  • Support firewall and security tools
  • Perform basic vulnerability analysis

Required Skills

  • Networking fundamentals
  • Linux basics
  • Wireshark
  • Firewall knowledge
  • Cybersecurity awareness

Cybersecurity Relevance

Network security is important because organizations depend on secure networks to protect data, prevent cyberattacks, and maintain business continuity.
